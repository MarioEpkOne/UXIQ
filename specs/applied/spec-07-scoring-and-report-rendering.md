## Goal

Implement two modules:
1. **`scorer.py`** ŌĆö computes per-tier star ratings and an overall weighted score from an `AuditReport`
2. **`report_renderer.py`** ŌĆö assembles a Markdown string from `AuditReport` + scores

These modules are deterministic and pure: given the same `AuditReport`, they always produce the same output. They do not call any API and do not raise exceptions.

### compute()

```python
from ui_analyzer.xml_parser import AuditReport

def compute(report: AuditReport) -> Scores:
    t1 = _compute_tier1_stars(report.tier1_findings)
    t2 = _compute_tier23_stars(report.tier2_findings)
    t3 = _compute_tier23_stars(report.tier3_findings)
    overall = round((t1 * 0.4) + (t2 * 0.35) + (t3 * 0.25), 1)
    return Scores(
        tier1=t1,
        tier2=t2,
        tier3=t3,
        tier4_flag_count=len(report.tier4_findings),
        overall=overall,
    )
```

### _compute_tier1_stars()

```python
def _compute_tier1_stars(findings) -> float:
    """% of Tier 1 checks that passed, mapped to 1ŌĆō5 stars.

    100% = 5Ōśģ, 80% = 4Ōśģ, 60% = 3Ōśģ, 40% = 2Ōśģ, <40% = 1Ōśģ
    No findings ŌåÆ 5Ōśģ (assume clean).
    """
    if not findings:
        return 5.0
    passes = sum(1 for f in findings if f.result == "PASS")
    total = len(findings)
    ratio = passes / total
    return max(1.0, round(ratio * 5, 1))
```

### _compute_tier23_stars()

```python
def _compute_tier23_stars(findings) -> float:
    """Avg severity inverted to stars.

    avg=0 ŌåÆ 5Ōśģ, avg=1 ŌåÆ 4Ōśģ, avg=2 ŌåÆ 2.5Ōśģ, avg=3 ŌåÆ 1Ōśģ
    No findings ŌåÆ 5Ōśģ (no issues found).
    """
    if not findings:
        return 5.0
    severities = [f.severity for f in findings if isinstance(f.severity, int)]
    if not severities:
        return 5.0
    avg = sum(severities) / len(severities)
    return max(1.0, round(5.0 - (avg * 1.5), 1))
```

### Star rendering helper

Used by `report_renderer.py`. Defined in `scorer.py` so it is independently testable:

```python
def stars_to_display(score: float) -> str:
    """Convert a 1.0ŌĆō5.0 score to a Unicode star string.

    E.g. 3.1 ŌåÆ "ŌśģŌśģŌśģŌśåŌśå"
    Rounds to nearest whole star for display.
    """
    filled = round(score)
    return "Ōśģ" * filled + "Ōśå" * (5 - filled)
```

### Tier 1 mode display

```python
tier1_mode_display = (
    "Authoritative (axe-core)"
    if axe_succeeded
    else "Estimated (visual)"

```

`axe_succeeded` is passed explicitly by the handler as `True` only when `run_axe()` returned an `AxeCoreResult`. When axe-core failed for a URL (`AxeFailure`), `axe_succeeded=False`, so the header correctly shows "Estimated (visual)" and is consistent with the per-finding `*(ESTIMATED)*` labels.

### Empty sections

If a tier has no findings:
```
*No issues found.*
```

If `inventory` or `structure_observation` is empty, omit that prose block ŌĆö do not print an empty block.

### parse_warnings passthrough

If `report.parse_warnings` is non-empty, append at the very end of the report (before the footer line):

## Constraints

- `report_renderer.py` must not import `anthropic`, `playwright`, or any I/O module.
- `scorer.py` must not import `anthropic`, `playwright`, or any I/O module.
- `render()` always returns a `str`. It never raises.
- `compute()` always returns `Scores`. It never raises.
- `stars_to_display()` rounds to the nearest whole number. 3.1 ŌåÆ 3 filled stars.
